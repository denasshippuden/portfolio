# Portfolio Andre Galante

Portfolio profissional criado com Next.js App Router, TypeScript, Tailwind CSS e Framer Motion.

## Rodando localmente

Instale as dependencias:

```bash
npm install
```

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

Depois acesse:

```text
http://localhost:3000
```

## Estrutura

- `app/`: paginas e layout com App Router
- `components/`: componentes reutilizaveis
- `data/`: dados editaveis de projetos, servicos e tecnologias
- `public/`: imagens e assets publicos

## Deploy

O projeto esta preparado para hospedagem futura na Vercel. Ao conectar o repositorio, a Vercel detecta Next.js automaticamente.
